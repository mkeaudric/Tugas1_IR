# Mesin Pencari Sederhana (Boolean & Tolerant Retrieval)

# Langkah-langkah menjalankan kode program: 
## 1. Clone Repositori
    git clone (https://github.com/mkeaudric/Tugas1_IR.git)
## 2. Buka folder proyek di code editor (Ex: Visual Studio Code, atau lainnya)
    cd Tugas1_IR
## 3. Kompilasi semua file .java dengan menggunakan 'run' atau kompilasi manual dengan cara: 
    javac *.java
## 4. Setelah kompilasi selesai, jalankan file main: (tidak perlu dilakukan jika menggunakan 'run' pada VS Code)
    java Main 
## 5. Program akan jalan, dan akan meminta untuk memasukkan Query (bisa satu kata atau lebih, atau satu kalimat)
## 6. Tekan tombol 'Enter' setelah sudah selesai memasukkan Query 
## 7. Tunggu sampai program selesai melakukan operasi pencarian dokumen relevan sesuai Query yang diberikan 
## 8. Setelah selesai, program akan menampilkan semua dokumen yang relevan dengan Query yang diberikan
